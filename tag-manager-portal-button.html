VinditScandit Tag Manager Portal Button

This is the portal button component to integrate into your vinditscandit.co.za website:

Default Style (Purple Gradient):

Access Tag Manager

Green Style:

NFC Tag Manager

Orange/Pink Style:

Manage NFC Tags

Integration Instructions:

  1. Copy the CSS code between the "TAG MANAGER PORTAL BUTTON" comments
  2. Add it to your vinditscandit.co.za stylesheet
  3. Copy one of the HTML button examples above
  4. Replace "https://your-tag-manager-domain.com" with your actual tag manager URL
  5. Place the button wherever you want it on your website

Customization Options:

  • Colors: Choose from default (purple), green, or orange/pink styles
  • Text: Change the button text to match your branding
  • Icon: Replace the SVG icon with your preferred icon
  • Size: Adjust padding and font-size in the CSS
